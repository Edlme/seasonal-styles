Outfitter for All Seasons!


"To everything, there is a season..."


Description of the web page

This single web page is created for seasonal wear outfitters, Eddie Browser. As the client want content that will change based on the season, including a look and feel, logo etc, I created the page using jQuery to give it the capability of changing with the seasons.

The page is created using jQuery in a way that promotes a unique look and feel for each of the four seasons; The following four areas of the page changes with the season:

  • A unique color background color is displayed for each season.
  • A unique logo is given for each of the season on the top-left corner of the page.
  • A unique item of clothing, is shown for each of the season.
  • A unique string of text identifying the season, is provided as a sub-heading.

I used switch to change items on the page to match the season. The click event handler that is foun in seasonal-styles.js file finds all of the anchor tags inside the indicated class(seasons).
